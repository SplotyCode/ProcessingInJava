# ProcessingInJava
Supported Functions:
* size(int width, int height)
* frameRate(short fps)
* fullScreen()
* noLoop()


* rect(float x, float y, float width, float height)
* triangle(float x1, float y1, float x2, float y2, float x3, float y3)
* point(float x, float y)
* point(float x, float y, float z)
* strokeWeight(int weight)
* line(float x1, float y1, float x2, float y2)
* elipse(float x, float y, float xRadius, float yRadius)
* stroke(int color)
* fill(int color)

Supported Variables:
* frameCount
* focused
* mousePressed
* mouseButton
* mouseButtonInt
* mouseX
* mouseY
* pmouseX
* pmouseY
* keyPressed
* key
* keyCode
* width
* height

Supported Events:
* setup()
* draw()

* mouseReleased()
* mousePressed()
* mouseClicked()
* mouseReleased(MouseClickEvent event)
* mousePressed(MouseClickEvent event)
* mouseClicked(MouseClickEvent event)

* mouseWheel(double yOffset)
* mouseDragged()
* mouseMoved()

* keyPressed()
* keyReleased()
* keyTyped()
* keyPressed(KeyBoardEvent event)
* keyReleased(KeyBoardEvent event)
* keyTyped(KeyBoardEvent event)
